## What not to open an issue?

To make features requests, cards with wrong image, decks errors or even wrongly final places for players...

For any of those reasons please enter in contact in a more informal way:

* [twitter](https://twitter.com/mtgmetaio) 
* [discord](https://discord.gg/3KaXbMQ)
* [email :email:](mailto:hello@mtgmeta.io)


## How to Report Bugs

Include the maximum of information possible, this can include:

* browser
* steps to reproduce the ocurrence
* screenshots

Then please open a [new issue](https://github.com/mtgmetaio/bugs-report/issues/new) and I will try to timely fix it.
